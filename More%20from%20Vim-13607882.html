More from Vim

March 15 2010, 4:57 PM  by Alexey Shamrin

Now is Vim time!

Lee Hinman wrote a blog post "How I develop Clojure with Vim". He's using Vim + screen + VimClojure + slime.vim. As usual, discussion can be found on Hacker News. Alternative scripts: vim plugin for screen/tmux integration; tslime.vim - a port of slime.vim to tmux.

By the way, what's tmux? According to the home page, tmux is just like screen, but with a modern implementation and BSD license.
