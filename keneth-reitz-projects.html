Keneth Reitz projects

June 7 2012, 5:52 AM  by Alexey Shamrin

Has just discovered Kenneth Reitz projects. Among them:

  • The Hitchhiker’s Guide to Python
  • HTTP Request & Response Service
  • Python Requests: HTTP for Humans
  • Tablib: Pythonic Tabular Datasets (with Excel output support, using packages from python-excel.org)
  • Legit - complementary command-line interface for Git, optimized for workflow simplicity
  • xerox: Copy + Paste in Python

By the way, he uses Pelican (static) weblog generator for his blog.
