PEG - Parsing Expression Grammar

March 21 2010, 6:57 PM  by Alexey Shamrin

Bryan Ford's "Parsing Expression Grammars" [pdf] is a new wave of language parsing grammars. Born in 2004. Some implementations:

  • PEG.js - parser generator for JavaScript with a cool demo (HN discussion).
  • OMeta - object-oriented language for pattern matching based on PEG. Several implementation, including JavaScript (with a JavaScript parser!).
  • another PEG parser for JavaScript language
