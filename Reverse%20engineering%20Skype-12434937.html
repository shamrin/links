Reverse engineering Skype

February 27 2010, 3:34 PM  by Alexey Shamrin

Fascinating (yes, old - 2006 - but still fascinating) presentation on reverse engineering Skype [pdf] (via HN).

In the process you can learn about different tools:

  • Scapy - a powerful interactive packet manipulation program
  • Rasta Ring 0 Debugger
  • PytStop is a debugging engine written in Python
  • plus lots of stuff on http://secdev.org/
